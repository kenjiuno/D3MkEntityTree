/*
 * D3MkEntityTree (Make Doom 3 entity list)
 */

//////////////////////////////////////////////////
// Name

D3MkEntityTree

//////////////////////////////////////////////////
// What's this

A tool run on Windows.
A tool for Doom 3 and Quake 4.

D3MkEntityTree will extract entity information from either Doom 3 or Quake 4 as html files.
