========================================================================
    スタティック ライブラリ : LibD3MkEntityTree プロジェクトの概要
========================================================================

AppWizard はユーザー用に LibD3MkEntityTree ライブラリ プロジェクトを作成しました。
ソース ファイルはプロジェクトの中には作成されていません。


LibD3MkEntityTree.vcproj
    これはアプリケーション ウィザードで生成された VC++ プロジェクトのメイン プロジェクト ファイルです。
    ファイルが生成された Visual C++ のバージョン情報が含まれています。また、 
    アプリケーション ウィザードで選択したプラットフォーム、構成およびプロジェクト機能に関する
    情報も含まれています。

/////////////////////////////////////////////////////////////////////////////
コンパイラとリンカ スイッチは MFC をサポートするように変更されました。
このプロジェクトで MFC ClassWizard を使用するには、プロジェクトにいくつかのファイルを追加する必要があります。 
追加ファイルには、"resource.h"、"LibD3MkEntityTree.rc"、および "LibD3MkEntityTree.h" が含まれます。
スタティック ライブラリに rc ファイルを追加するとDll または Exe には rc ファイルは 1 つしか存在できないという制限があるために、問題が生じる可能性があります。
この問題を解決するには、ライブラリの .rc ファイルを 
親プロジェクトの .rc ファイルに含めてください。
